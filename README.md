# Queens Of Change Foundation 🌸

A simple, responsive, and creative webpage designed for the **Queens Of Change Foundation**.  
This project highlights the NGO’s mission to empower women and transform communities.

---

## ✨ Features
- **[NGO Name](ca://s?q=Queens_Of_Change_Foundation)** prominently displayed in the header
- Inspiring **[tagline](ca://s?q=Empowering_Women_Tagline)** with subtle animations
- **[About section](ca://s?q=About_Queens_Of_Change_Foundation)** describing the mission
- One **[image](ca://s?q=Queens_Of_Change_Foundation_Image)** (replace with your own)
- Call-to-action **[button](ca://s?q=Volunteer_or_Join_Us_Button)** (Join Us / Volunteer / Donate)
- Clean **[footer](ca://s?q=Queens_Of_Change_Foundation_Footer)** with copyright
- Responsive design for mobile and desktop
- Smooth animations for a modern look
